How to open PingStats

This build is not notarized by Apple, so macOS blocks it the first time with “Apple could not verify…”. That warning cannot be removed without an Apple Developer account. Allow the app once in Privacy & Security.

  1. Drag PingStats to the Applications folder.
  2. Open PingStats from Applications once. When the warning appears, click Done (this makes the allow control show up in Settings).
  3. Open Privacy & Security settings:
    Open Privacy & Security

    If that button does nothing, try this alternate link, or open System Settings → Privacy & Security yourself.

  4. Scroll down to the message about PingStats and click Open Anyway. Confirm if asked.
  5. PingStats should open and appear in the menu bar. Later launches work normally.
Tip: There are no install scripts on purpose — double-clicked scripts are blocked the same way as the app. Allowing the app once in Privacy & Security is the simplest path.
